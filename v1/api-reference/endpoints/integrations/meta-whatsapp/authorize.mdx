---
title: "Authorize Meta WhatsApp"
api: "GET /v1/api/meta-whatsapp/authorize"
description: "Get the Meta OAuth authorization URL to connect your WhatsApp Business Account."
---

## Headers
<ParamField header="x-workspace-id" type="string" required>
Workspace identifier for the API.
</ParamField>

## Response Fields

<ResponseField name="url" type="string">
The Meta OAuth authorization URL. Redirect the user to this URL to authorize the WhatsApp Business connection.
</ResponseField>

```json
{
  "url": "https://www.facebook.com/v18.0/dialog/oauth?client_id=...&redirect_uri=...&scope=...&state=..."
}
```
</ResponseExample>
