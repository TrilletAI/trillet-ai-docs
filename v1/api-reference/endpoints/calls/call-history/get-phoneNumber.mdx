---
title: "Get Phone Numbers"
api: "GET /v1/api/twilio/user-phone-numbers"
description: "Retrieve a list of phone numbers associated with the user."
---

### Headers

<ParamField header="x-workspace-id" type="string" required>
  Workspace identifier for the API.
</ParamField>

### Response Fields

<ResponseField name="phoneNumbers" type="array">
  An array of phone numbers associated with the user, in E.164 format.
</ResponseField>
