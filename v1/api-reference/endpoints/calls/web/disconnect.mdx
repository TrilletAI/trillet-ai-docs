---
title: "Disconnect a Web Call"
api: "POST /v1/api/calls/web/disconnect"
description: "End an ongoing web-based call session."
---

End an active web-based call session. This API notifies the server that the user is done.

### Headers  

<ParamField header="x-api-key" type="string" required>  
API key used for authenticating requests to the API.
</ParamField>  


<ParamField header="x-workspace-id" type="string" required>
  Workspace identifier for the API.
</ParamField>


### Request Body

<ParamField body="roomName" type="string" required>
  The room name for the ongoing LiveKit session.
</ParamField>

<ResponseExample>
```json
{
  "status": "success",
  "message": "Session ended successfully"
}
```
</ResponseExample>
